webas400app / kidakorn — Basic Build

IWS Basic Setup: Fetch & Display + QR Code Generator

A minimal build for a blank shared folder: pull data from IWS, show it in a table, and generate a QR code for any identifier field (operator code, process code, etc.).

ไกด์แบบพื้นฐานสำหรับโฟลเดอร์แชร์เปล่าๆ: ดึงข้อมูลจาก IWS มาแสดงเป็นตาราง และสร้าง QR code จาก field ที่เป็นรหัส/identifier เช่น รหัสพนักงาน (Operator code) หรือรหัสกระบวนการ (Process code)

js/config.js
ตั้งค่า config.js
const ENVIRONMENTS = {
  local: { BASE_URL: "/api/web/serveices" },
  production: { BASE_URL: "/api/web/serveices" }   // 🔴 แก้ตรงนี้ตอนขึ้นจริงถ้า path ต่างจาก dev
};

function detectEnvironment() {
  const host = window.location.hostname;
  return (host === "localhost" || host === "127.0.0.1") ? "local" : "production";
}

const IWS_CONFIG = {
  BASE_URL: ENVIRONMENTS[detectEnvironment()].BASE_URL,
  ENDPOINTS: {
    GET_DATA: "/yourservice/GET_DATA"   // 🔴 แก้ path ให้ตรงกับที่แอดมิน deploy จริง
  }
};
ภาษาไทย: ตัวอย่างพื้นฐานมี endpoint เดียว (GET) เพราะแค่ดึงมาแสดง ไม่มีการบันทึกกลับ

js/api.js
ตั้งค่า api.js
async function handleResponse(res) {
  const raw = await res.text();
  if (!res.ok) throw new Error(`HTTP ${res.status}: ${raw}`);
  return JSON.parse(raw);
}

const IWS_API = {
  async getData() {
    const res = await fetch(IWS_CONFIG.BASE_URL + IWS_CONFIG.ENDPOINTS.GET_DATA, {
      headers: { "Accept": "application/json" }
    });
    return handleResponse(res);
  }
};
4
Add the QR code library
เพิ่มไลบรารีสร้าง QR code

Easiest way for development — load from CDN:

<script src="https://cdnjs.cloudflare.com/ajax/libs/qrcodejs/1.0.0/qrcode.min.js"></script>
🔴 PRODUCTION CHANGE: If the production server/network has no internet access (common on internal AS400/factory networks), this CDN link will fail silently and QR codes won't generate. Download qrcode.min.js once, save it to js/vendor/qrcode.min.js in your shared folder, and change the script tag to: 
<script src="js/vendor/qrcode.min.js"></script>
ภาษาไทย: เครือข่ายโรงงาน/production มักไม่มี internet ออกนอก ถ้าใช้ CDN ตรงๆ QR จะไม่ขึ้นเลยตอนขึ้นจริง — ให้โหลดไฟล์ไลบรารีมาเก็บไว้ในโฟลเดอร์ (self-host) แทน ปลอดภัยกว่าเยอะ
5
index.html — fetch, display, generate QR
index.html — ดึง แสดง สร้าง QR
<!DOCTYPE html>
<html lang="th">
<head>
<meta charset="UTF-8">
<meta name="color-scheme" content="light">
<title>Data + QR</title>
<script src="js/vendor/qrcode.min.js"></script> <!-- 🔴 เปลี่ยนเป็น local path ตอน production -->
</head>
<body>
  <h1>Data List</h1>
  <button id="loadBtn">Load Data</button>
  <table id="dataTable"></table>

  <div id="qrModal" style="display:none;">
    <div id="qrcode"></div>
    <p id="qrLabel"></p>
    <button onclick="document.getElementById('qrModal').style.display='none'">Close</button>
  </div>

  <script src="js/config.js?v=1"></script>
  <script src="js/api.js?v=1"></script>
  <script>
    let qrInstance = null;

    function showQR(value, label) {
      document.getElementById('qrModal').style.display = 'block';
      document.getElementById('qrLabel').textContent = label + ': ' + value;
      document.getElementById('qrcode').innerHTML = '';
      qrInstance = new QRCode(document.getElementById('qrcode'), {
        text: String(value),
        width: 180,
        height: 180
      });
    }

    async function loadData() {
      const data = await IWS_API.getData();
      const rows = Array.isArray(data) ? data : [data];
      const table = document.getElementById('dataTable');
      const cols = Object.keys(rows[0] || {});

      let html = '<tr>' + cols.map(c => `<th>${c}</th>`).join('') + '<th>QR</th></tr>';
      rows.forEach(r => {
        html += '<tr>' + cols.map(c => `<td>${r[c]}</td>`).join('');
        // สมมติ column แรกเป็น identifier ที่จะเจนเป็น QR (เช่น operator code / process code)
        html += `<td><button onclick="showQR('${r[cols[0]]}', '${cols[0]}')">Generate QR</button></td></tr>`;
      });
      table.innerHTML = html;
    }

    document.getElementById('loadBtn').addEventListener('click', loadData);
  </script>
</body>
</html>
ภาษาไทย: ตัวอย่างนี้สมมติว่า column แรกของข้อมูลคือ identifier (เช่น Operator code) ถ้าอยากเจน QR จาก field อื่น แก้ที่ cols[0] ให้ชี้ไปที่ field ที่ต้องการแทน

web.config — same 2 additions
web.config — เพิ่ม 2 อย่างเหมือนเดิม
<rule name="YourExistingRule" stopProcessing="true">
  <match url="^(.*)$" />
  <conditions logicalGrouping="MatchAll">
    <add input="{REQUEST_FILENAME}" matchType="IsFile" negate="true" />
    <add input="{REQUEST_FILENAME}" matchType="IsDirectory" negate="true" />
  </conditions>
  <action type="Rewrite" url="..." />
</rule>
ภาษาไทย: ต้องมี conditions นี้เสมอ ไม่งั้น rewrite rule เดิมจะทับไฟล์ .js และไฟล์ QR library ที่ self-host ไว้ไม่ให้โหลดได้
