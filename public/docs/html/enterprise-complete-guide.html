02 — Skill Requirements

สกิลและความรู้
ที่ต้องใช้แต่ละโปรเจ็ค

แต่ละโปรเจ็คต้องการสกิลเทคนิค ความรู้เชิงธุรกิจ และแนวทางสร้างรายได้ที่แตกต่างกัน

ERP + บัญชี-การเงิน
Enterprise Resource Planning / Accounting
WebDesktop
เทคนิค
  • Database design ซับซ้อน (transaction integrity)
  • Modular architecture, REST/GraphQL API
  • Reporting engine (PDF/Excel export)
  • Audit log ห้ามแก้ไขย้อนหลัง
  • Decimal arithmetic (ไม่ใช้ float)
ความรู้ธุรกิจ
  • หลักบัญชีคู่ (double-entry bookkeeping)
  • ผังบัญชี มาตรฐาน TFRS/IFRS
  • ภาษีมูลค่าเพิ่ม ภาษีหัก ณ ที่จ่าย
  • Workflow PR → PO → GR → Invoice
สร้างรายได้
  • License รายปี หรือ SaaS subscription
  • คิดค่า implementation/customization แยก
  • ต้นทุน support สูง ต้องคำนวณใน pricing
HRM / Payroll
Human Resource Management
WebMobile
เทคนิค
  • Cron job สำหรับคำนวณเงินเดือนรอบบิล
  • Mobile: GPS / face recognition ลงเวลา
  • PDF generation สลิปเงินเดือน
  • State machine (ลา: Pending → Approved)
ความรู้ธุรกิจ
  • กฎหมายแรงงาน ประกันสังคม กองทุน
  • สูตรคำนวณภาษีเงินได้บุคคลธรรมดา
  • PDPA — ข้อมูลพนักงานอ่อนไหวมาก
สร้างรายได้
  • per employee/เดือน เป็นมาตรฐานตลาด
  • เพิ่มมูลค่าด้วย mobile check-in app
Logistics + WMS
Tracking / Warehouse Management
WebMobile
เทคนิค
  • GPS tracking, WebSocket real-time
  • Google Maps / Mapbox API, geofencing
  • Barcode/QR scanning บน mobile
  • Concurrency control เมื่อหลายคนเบิกพร้อมกัน
ความรู้ธุรกิจ
  • Flow คลัง: รับ-จัดเก็บ-หยิบ-แพ็ค-ส่ง
  • Lot/Serial number, วันหมดอายุ
  • SLA การจัดส่ง, last-mile delivery
สร้างรายได้
  • per vehicle หรือ per trip (usage-based)
  • ขาย module เสริม ERP หรือแยกให้ 3PL
CRM + POS + E-commerce
Sales & Customer Systems
WebMobileDesktop
เทคนิค
  • Pipeline/funnel data model
  • Offline-first POS + sync strategy
  • Payment gateway + webhook (Omise/Stripe)
  • SEO + Core Web Vitals (E-commerce)
  • Inventory lock ขณะชำระเงิน
ความรู้ธุรกิจ
  • Sales funnel, lead scoring
  • กฎหมายใบกำกับภาษีอย่างย่อ/เต็ม
  • Digital marketing พื้นฐาน
  • ส่วนลด โปรโมชั่น แต้มสะสม
สร้างรายได้
  • CRM: per seat/เดือน
  • POS: hardware bundle + subscription
  • E-commerce: % GMV หรือรายเดือน
BI + DMS + IAM
Data / Document / Security Systems
Web
เทคนิค
  • ETL pipeline ดึงข้อมูลหลายระบบ
  • S3/MinIO file storage, versioning
  • OAuth2/OIDC/SAML สำหรับ SSO
  • RBAC/ABAC, rate limiting, encryption
ความรู้ธุรกิจ
  • KPI design ตรงเป้าหมายธุรกิจ
  • มาตรฐานการจัดเก็บเอกสารตามกฎหมาย
  • ISO 27001 พื้นฐาน, PDPA
สร้างรายได้
  • BI: add-on premium tier
  • DMS: per GB storage
  • IAM: B2B API infrastructure service
