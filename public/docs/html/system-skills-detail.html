ระบบบริหารทรัพยากรองค์กร

ระบบ ERP

Enterprise Resource Planning

WebDesktop

เทคนิค

  • Database design ระดับซับซ้อน (normalization, transaction integrity)
  • Backend: Node.js/Java/.NET + REST/GraphQL API
  • Module architecture — แยกระบบย่อยแต่เชื่อมข้อมูลกันได้ (microservices หรือ modular monolith)
  • Reporting engine (PDF/Excel export, pivot table)

ความรู้ธุรกิจ

  • หลักการบัญชีคู่ (double-entry bookkeeping)
  • Workflow อนุมัติเอกสาร (PR/PO/GR) ขององค์กรจริง
  • มาตรฐานภาษีและเอกสารทางการเงินในประเทศที่จะขาย

สร้างรายได้

  • ขายแบบ License รายปี หรือ SaaS subscription ตามจำนวนผู้ใช้
  • คิดค่า implementation/customization แยกจากค่าซอฟต์แวร์
  • ต้นทุน support สูง ต้องคำนวณใน pricing

ระบบบัญชี-การเงิน

Accounting & Finance

WebDesktop

เทคนิค

  • การคำนวณทางการเงินแม่นยำ (ใช้ Decimal ไม่ใช่ float)
  • Audit log / ห้ามแก้ไขข้อมูลย้อนหลังโดยไม่มีร่องรอย
  • เชื่อมต่อ Bank API หรือไฟล์ statement (CSV/MT940)

ความรู้ธุรกิจ

  • มาตรฐานบัญชี (TFRS/IFRS) และผังบัญชี
  • ภาษีมูลค่าเพิ่ม ภาษีหัก ณ ที่จ่าย
  • งบการเงิน 3 รูปแบบหลัก: งบดุล งบกำไรขาดทุน กระแสเงินสด

สร้างรายได้

  • เหมาะกับขาย SaaS รายเดือนให้ SME
  • เพิ่มมูลค่าด้วยฟีเจอร์เชื่อมต่อระบบยื่นภาษีออนไลน์

ระบบจัดซื้อจัดจ้าง

Procurement

WebMobile

เทคนิค

  • Approval workflow engine (multi-level, conditional routing)
  • Notification system (email/push) สำหรับขั้นตอนอนุมัติ
  • State machine ของสถานะเอกสาร

ความรู้ธุรกิจ

  • กระบวนการจัดซื้อ: PR → RFQ → PO → GR → Invoice
  • การเปรียบเทียบผู้ขายและเงื่อนไขสัญญา

สร้างรายได้

  • ขายเป็น add-on module ผูกกับ ERP
  • คิดค่าบริการตามปริมาณธุรกรรม (transaction-based)

ระบบ HRM / HRIS

Human Resource Management

WebMobile

เทคนิค

  • Cron job/scheduler สำหรับคำนวณเงินเดือนรอบบิล
  • Mobile: GPS/face recognition สำหรับลงเวลา
  • การจัดการไฟล์เอกสาร (สลิปเงินเดือน, สัญญาจ้าง)

ความรู้ธุรกิจ

  • กฎหมายแรงงาน, ประกันสังคม, กองทุนสำรองเลี้ยงชีพ
  • โครงสร้างเงินเดือนและสูตรคำนวณภาษี ณ ที่จ่าย
  • PDPA — ข้อมูลพนักงานเป็นข้อมูลส่วนบุคคลอ่อนไหว

สร้างรายได้

  • Pricing แบบ per-employee/เดือน เป็นมาตรฐานตลาด

ระบบติดตามและปฏิบัติการ

ระบบติดตามงาน

Task / Project Tracking

WebDesktopMobile

เทคนิค

  • Real-time sync (WebSocket) เพื่ออัปเดตสถานะงานทันที
  • Drag-and-drop UI (Kanban board)
  • Offline-first สำหรับ mobile

ความรู้ธุรกิจ

  • หลัก Agile/Scrum/Kanban
  • การวัดผลงานด้วย KPI/OKR

สร้างรายได้

  • Freemium model — ฟรีสำหรับทีมเล็ก เก็บเงินทีมใหญ่/ฟีเจอร์ขั้นสูง
  • ตลาดแข่งขันสูง ต้องหา niche เฉพาะอุตสาหกรรม

ระบบติดตามสินค้า/ขนส่ง

Logistics Tracking

WebMobile

เทคนิค

  • Google Maps/Mapbox API, geofencing
  • GPS tracking แบบ real-time, การจัดการ background location บน mobile
  • Route optimization algorithm พื้นฐาน

ความรู้ธุรกิจ

  • กระบวนการขนส่งและ last-mile delivery
  • เงื่อนไข SLA การจัดส่ง

สร้างรายได้

  • คิดค่าบริการตามจำนวนพัสดุ/เที่ยววิ่ง (usage-based)

ระบบบริหารคลังสินค้า

Warehouse Management (WMS)

WebMobile

เทคนิค

  • Barcode/QR scanning integration บน mobile
  • การจัดการ concurrency เมื่อหลายคนเบิก-รับพร้อมกัน
  • Inventory valuation method (FIFO/LIFO/Avg cost)

ความรู้ธุรกิจ

  • Flow คลังสินค้า: รับเข้า-จัดเก็บ-หยิบ-แพ็ค-ส่งออก
  • การจัดการ lot/serial number, วันหมดอายุ

สร้างรายได้

  • ขายเป็น module เสริม ERP หรือแยกขาย SaaS ให้ 3PL

ระบบบำรุงรักษา

Maintenance / CMMS

WebMobile

เทคนิค

  • Scheduler สำหรับแผนบำรุงรักษาเชิงป้องกัน (Preventive Maintenance)
  • QR code ติดเครื่องจักร เชื่อมประวัติการซ่อม

ความรู้ธุรกิจ

  • หลัก Preventive vs Corrective Maintenance
  • พื้นฐานเครื่องจักร/อุปกรณ์ในอุตสาหกรรมเป้าหมาย

สร้างรายได้

  • เหมาะกับขายเฉพาะกลุ่มโรงงาน/อุตสาหกรรม (vertical SaaS)

ระบบลูกค้าและการขาย

ระบบ CRM

Customer Relationship Management

WebMobile

เทคนิค

  • Pipeline/funnel data model
  • Email/SMS integration (SendGrid, Twilio)
  • Tagging และ segmentation engine

ความรู้ธุรกิจ

  • Sales funnel และ lead scoring
  • กลยุทธ์ retention/upsell

สร้างรายได้

  • SaaS per-seat pricing เป็นมาตรฐาน
  • เพิ่มรายได้ด้วย integration กับ marketing tools

ระบบขายหน้าร้าน

Point of Sale (POS)

DesktopMobile

เทคนิค

  • Offline-first — ต้องขายได้แม้เน็ตหลุด แล้ว sync ทีหลัง
  • เชื่อมต่อฮาร์ดแวร์: เครื่องพิมพ์ใบเสร็จ, เครื่องรูดบัตร, ลิ้นชักเงินสด
  • Payment gateway integration (พร้อมเพย์/บัตรเครดิต)

ความรู้ธุรกิจ

  • กฎหมายใบกำกับภาษีอย่างย่อ/เต็มรูป
  • การจัดการส่วนลด โปรโมชั่น แต้มสะสม

สร้างรายได้

  • ขายฮาร์ดแวร์+ซอฟต์แวร์เป็นชุด หรือเก็บค่า transaction fee

ระบบ E-commerce

Online Store

WebMobile

เทคนิค

  • Payment gateway + webhook handling (Omise, Stripe, 2C2P)
  • SEO และ performance optimization (Core Web Vitals)
  • Cart/checkout state management, inventory lock ขณะชำระเงิน

ความรู้ธุรกิจ

  • Digital marketing พื้นฐาน (conversion rate, cart abandonment)
  • กฎหมายคุ้มครองผู้บริโภคออนไลน์

สร้างรายได้

  • ขายแพลตฟอร์มให้ร้านค้า (เก็บ % GMV หรือรายเดือน)
  • หรือสร้างร้านขายเองแล้วทำกำไรจากสินค้าโดยตรง

ระบบบริการลูกค้า

Helpdesk / Ticketing

WebMobile

เทคนิค

  • SLA timer/escalation logic
  • Live chat (WebSocket) และ omnichannel integration (LINE, email)

ความรู้ธุรกิจ

  • มาตรฐานการวัดผล CSAT/SLA

สร้างรายได้

  • SaaS per-agent pricing

ระบบข้อมูลและการตัดสินใจ

ระบบรายงานผู้บริหาร

Business Intelligence / Dashboard

WebDesktop

เทคนิค

  • ETL pipeline พื้นฐาน (ดึง-แปลง-โหลดข้อมูลจากหลายระบบ)
  • Data visualization library (Recharts, D3, Power BI embed)
  • Query optimization สำหรับข้อมูลปริมาณมาก

ความรู้ธุรกิจ

  • การออกแบบ KPI ที่ตรงกับเป้าหมายธุรกิจจริง
  • หลักสถิติพื้นฐานสำหรับตีความข้อมูล

สร้างรายได้

  • ขายเป็น add-on premium tier ของระบบอื่น

ระบบจัดการเอกสาร

Document Management (DMS)

WebDesktop

เทคนิค

  • File storage (S3-compatible) + version control ของไฟล์
  • Full-text search (Elasticsearch/OpenSearch)
  • OCR สำหรับเอกสารสแกน

ความรู้ธุรกิจ

  • มาตรฐานการจัดเก็บเอกสารและระยะเวลาเก็บตามกฎหมาย

สร้างรายได้

  • Pricing ตามพื้นที่จัดเก็บ (storage-based)

ระบบควบคุมสิทธิ์การเข้าถึง

Identity & Access Management

Web

เทคนิค

  • OAuth2/OpenID Connect, SAML สำหรับ SSO
  • RBAC/ABAC design — โครงสร้างสิทธิ์ที่ยืดหยุ่น
  • Security hardening: rate limiting, encryption at rest

ความรู้ธุรกิจ

  • มาตรฐานความปลอดภัยข้อมูล (ISO 27001 พื้นฐาน)

สร้างรายได้

  • ขายเป็น infrastructure service ให้ระบบอื่นเชื่อมต่อ (B2B API)
