webas400app / kidakorn

ดึงข้อมูลจริงจาก AS400 — วันนี้ทำแค่นี้พอ

โฟกัสแค่ config.js + api.js ให้ดึงข้อมูลจริงจาก endpoint scanner/minpop ได้ก่อน ส่วนอื่นค่อยว่ากันทีหลัง

1
เช็ค endpoint จริงก่อน

เปิด browser ไปที่ URL ที่มีอยู่แล้ว เช็คว่ายังได้ JSON กลับมาอยู่ไหม

http://as400webappdev/api/web/serveices/scanner/minpop?empid=25066
ถ้าได้ JSON กลับมา แปลว่า endpoint ใช้งานได้ ไปข้อ 2 ต่อได้เลย
2
แก้ js/config.js (2 บรรทัด)
const IWS_CONFIG = {
  BASE_URL: "/api/web/serveices",   // ให้ตรงกับ path จริงที่เช็คในข้อ 1
  ENDPOINTS: {
    GET_DATA: "/scanner/minpop"     // path ต่อจาก BASE_URL (ไม่รวม query string ?empid=)
  }
};
ภาษาไทย: แก้แค่ 2 ค่านี้พอ ที่เหลือในไฟล์ config.js ไม่ต้องแตะ
3
แก้ js/api.js ให้รองรับ query param

ของเดิมยังไม่รองรับ ?empid= ต้องแก้ให้รับค่าเข้ามาแล้วต่อท้าย URL

const IWS_API = {
  async getData(empid) {
    const res = await fetch(
      IWS_CONFIG.BASE_URL + IWS_CONFIG.ENDPOINTS.GET_DATA + "?empid=" + empid,
      { headers: { "Accept": "application/json" } }
    );
    return handleResponse(res);
  }
};
4
แก้ js/app.js ให้ส่ง empid เข้าไป

ตอนนี้ hardcode รหัสทดสอบไปก่อน ค่อยเปลี่ยนเป็นรับจาก input ทีหลัง

async function loadData() {
  setStatus('pending', 'กำลังโหลด...');
  try {
    const data = await IWS_API.getData("25066");   // ใส่รหัสทดสอบไปก่อน
    renderRows(Array.isArray(data) ? data : [data]);
    setStatus('ok', 'โหลดสำเร็จ (จาก IWS จริง)');
  } catch (err) {
    setStatus('bad', 'ผิดพลาด');
    document.getElementById('dataTable').innerHTML = `<tr><td>Error: ${err.message}</td></tr>`;
  }
}
5
เปิดผ่าน URL จริงเท่านั้น
http://as400webappdev/...
ห้าม double-click เปิดไฟล์ตรงๆ ไม่งั้น relative path (/api/...) จะเรียกไม่ติด เพราะไม่ได้เป็น same-origin กับ IWS
6
เปิด DevTools → Console เช็ค 2 อย่าง
  • เห็น [IWS_CONFIG] ... = config โหลดถูก
  • กด "Load Data" แล้วดู error (ถ้ามี)
Errorความหมาย
404path ผิด — เช็ก BASE_URL/ENDPOINTS อีกที
CORS errorเรียกข้าม domain — เช็กว่าเปิดผ่าน as400webappdev จริงไหม
500ฝั่ง backend มีปัญหา — แจ้งแอดมินเช็ก joblog
ติดตรงไหน: ถ่ายรูป error ใน Console ส่งมาได้เลย จะได้ช่วยดูให้ทัน
