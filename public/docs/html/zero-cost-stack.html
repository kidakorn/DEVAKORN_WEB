Containerize & Deploy (ภายหลัง)

local container ก่อน, cloud ทีหลังถ้าต้องการ
แนวทางหลัก: พัฒนาและรันด้วย Docker บนเครื่องตัวเอง ก่อนเสมอ — ครบทุก database, service ต่างๆ รันแบบ local container ทั้งหมด ไม่ต้องพึ่ง cloud account ระหว่างพัฒนา เครื่องมือ cloud ด้านล่างเป็นตัวเลือกสำหรับ "ตอนต้องการ deploy จริง" เท่านั้น ไม่ใช่ขั้นตอนบังคับ

Open Source

แนะนำLocal DevProduction-like
docs.docker.com/engine
